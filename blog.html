启动AWS EC2的一个实例

申请AWS帐号

需要一个能够美元支付信用卡,为验证信用卡会先收$1, 据说一个月后退回. 可以免费在限额内使用一年, 注意 超出限额仍然要付费.

启动EC2的实例

  • 选可免费的amazon linux机器.
  • 选免费套餐类型, 审核并启动之.
  • 生成或选择已有密钥对, 下载生成的密钥文件(如I1.pem), 用于登录管理以及上传数据.

ssh登录服务器, 并管理

  1. 查询实例的IP地址(如a.b.c.d)
  2. 登录: ssh -i I1.pem ubuntu@a.b.c.d
  3. 在服务器上, 安装nginx web server.
  4. 在服務器上, 配置nginx.conf文件, web的根目录, 证书位置.
  5. 上传file到服務器 : scp -i I1.pem file ubuntu@a.b.c.d:

证书

  1. 获得证书(免费从阿里云等等, 或购买)
  2. 下载证书文件(*.key, *.pem, 记录值), 并上传到AWS EC2服务器,
